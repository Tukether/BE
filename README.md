# Tukether Backend

한국공학대학교 커뮤니티 백엔드 - Django REST Framework

## 기술 스택

- **Backend**: Django 5.1.3
- **API**: Django REST Framework 3.14.0
- **Database**: MySQL
- **Authentication**: JWT (djangorestframework-simplejwt)
- **Server**: Gunicorn

## 팀원 가이드
**주의**: `.env` 파일은 절대 Git에 커밋하지 마!
